### Sorting Benchmark (HW-04)

Короткий бенчмарк сортувань: генерує масиви різних типів і міряє час роботи алгоритмів сортування. Виводить таблицю з часом виконання для кожної комбінації.

### Як запустити

```bash
python3 main.py
```

Налаштування за замовчуванням (зашиті в коді):
- *sizes (розмір датасетів)*: **1000, 3000**
- *dists (розподіл даних)*:
  - **random** - випадкові значення;
  - **sorted** - відсортовані за зростанням
  - **reversed** - відсортовані у зворотньому порядку;
- *algos (алгоритми сортування)*:
  - **timsort** (вбудований `sorted`);
  - **merge** (сортування злиттям);
  - **insertion** (сортування вставками)
- *seed*: **42** (для відтворюваності генерації випадкових даних)

### Що відбувається
- Генерація даних.
- Запуск кожного алгоритму на кожному наборі даних.
- Вимір часу за допомогою `timeit` (кілька повторів) і вивід найкращого середнього часу.
- Результати показуються у табличному форматі по розміру, типу даних і алгоритму.

### Структура
- `datasets.py` — генератори даних, реєстр `DATASETS`
- `algorithms.py` — алгоритми сортування, реєстр `ALGORITHMS`
- `main.py` — бенчмарк і форматований вивід результатів
